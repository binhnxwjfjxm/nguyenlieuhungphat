# Website Nguyên Liệu Hưng Phát

Sprint 1 tập trung vào giao diện responsive:

- Design system nền kem, nhấn nâu và xanh lá
- Header desktop và menu mobile
- Mobile UI theo hướng mẫu 03
- Bottom navigation trên điện thoại
- Hero, danh mục, card sản phẩm và footer
- Motion, trạng thái nhấn và phản hồi rung trên thiết bị hỗ trợ

## Chạy local

```bash
npm install
npm run dev
```

## Kiểm tra build

```bash
npm run build
```
